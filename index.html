My Projects are

Product Online System

The Product Online System is a web application where admins and customers can securely register and log in. Admins manage product details such as adding, updating, or deleting products, while customers can view products, make purchases, and manage their profiles.

Quiz Competition System

The Quiz Competition System is a web application where admins and participants can securely register and log in. Admins can create and manage quizzes by adding questions and setting time limits, while participants can take quizzes, view their scores, and track their performance.

Food Ordering System

The Online Food Ordering System is a web application where admins manage food items—including adding, updating, and deleting products—while customers can browse menus, place orders, track status, and manage profiles. The system ensures secure login and a user-friendly interface for efficient online food ordering.

Bank Management System

The Bank Management System is a web application that enables users to register, log in, check balances, deposit/withdraw funds, transfer money, and view bank details. Built with HTML, CSS, JavaScript, Java (MVC), MySQL, and deployed on Apache Tomcat.

Management System

The Attendance Management System is a web application where students and employees can securely register and log in. Employees manage student details and attendance, while students view their profiles and attendance history.
